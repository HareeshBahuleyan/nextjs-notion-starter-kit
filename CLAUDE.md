# Claude Code Guidelines

## Pull Requests

Always create pull requests against the user's own fork:
- Repo: `HareeshBahuleyan/nextjs-notion-starter-kit`
- Use `gh pr create --repo HareeshBahuleyan/nextjs-notion-starter-kit` — never omit `--repo`, as `gh` will otherwise default to the upstream `transitive-bullshit/nextjs-notion-starter-kit`.
